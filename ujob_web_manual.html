1. 职位管理

发布、刷新、关闭与编辑职位

1.1发布职位

  1. 进入【职位管理】→【社员】。
  2. 点击【+发布社员职位】。
  3. 填写职位信息。
  4. 点击【保存】或【提交审核】；审核通过后自动发布。
已保存未提交的职位，可在【职位管理】→【未提出】状态列表中查看。

1.2职位刷新

  • 单独刷新：勾选职位后点击【刷新】。
  • 批量刷新：勾选【全选】后点击【刷新】。
批量刷新仅作用于当前页已选中的职位。

1.3关闭 / 编辑职位

  • 关闭职位：点击【关闭职位】→【确定】。
  • 编辑职位：点击【编辑】→ 修改信息 →【保存】或【提交审核】。

2. 寻找人才

智能匹配与筛选候选人

2.1智能匹配

  1. 进入【人才推荐】，选择职位类型，系统会自动匹配人才。
  2. 人才列表支持【推荐】和【最新】两种显示方式；【最新】按用户活跃时间排序。
  3. 【已收藏】列表显示已收藏在线简历的人才。

2.2筛选人才

  • 支持按日语、经验、技能、学历等条件多选筛选。
  • 已选条件会以标签展示，可单独删除，也可一键清空。

3. 沟通约聊

查看简历、聊天、约面试与发送职位

3.1查看简历与聊天

  1. 进入【人才推荐】→【社员】/【时给】/【IT求人】。
  2. 点击人才卡片查看在线简历。
  3. 点击【打招呼】直接发起聊天。
  4. 在【在线沟通】中查看历史聊天记录。

3.2交换信息 / 要简历 / 约面试

  • 【给微信】：向候选人发送微信号。
  • 【要简历】：请求候选人发送简历。
  • 【约面试】：选择时间后发送面试邀请。

3.3发送职位信息

点击【选择职位】→ 选择要发送的职位 →【确定】。

3.4自定义打招呼语

点击【常】→【+追加】→ 输入内容（500字内）→【确定】。

4. 简历中心

下载附件简历与查看在线简历

4.1附件简历

  1. 进入【简历中心】。
  2. 点击【附件简历】。
  3. 点击【下载简历】。

4.2在线简历

  1. 进入【简历中心】，切换到【在线简历】。
  2. 点击人才卡片查看在线简历。

5. 公司管理

企业信息维护与认证

5.1企业信息

  1. 进入【公司管理】，点击【编辑】。
  2. 点击头像卡片上传企业LOGO。
  3. 修改公司简称、公司行业、人员规模、公司描述、公司地址、公司官网等信息。
  4. 点击相册卡片上传公司照片。
如需修改公司全称，请联系专属客服。

5.2企业认证

  1. 在认证区域点击卡片上传营业执照。
  2. 点击【确定】提交审核。
已认证成功的企业，如需重新上传认证信息，请联系专属客服。
